# Contact Management Express App

## Overview

This Express app is designed for managing contacts. It provides a RESTful API with CRUD operations for handling contact information. The app is built on Node.js and uses MongoDB for data storage and uses Mongoose ODM. Authentication is implemented using JSON Web Tokens (JWT) for secure access.

## Features

- **Contact Management**: Perform CRUD operations on contacts (Create, Read, Update, Delete).
- **MongoDB Integration**: Utilizes MongoDB as a database to store contact information.
- **Authentication**: Secures endpoints using JWT tokens.

## Prerequisites

Before running the application, ensure the following software is installed on your machine:

- node v18.x

## Installation

You can install the required dependencies using `npm`. Just run:

```
$ npm install
```
## Configuration
Before you run, you need to configure the .env file parameters by taking .env.sample file as a reference.

To start the server, just run:

```
$npm start
```
